## Ejemplo 4: Uso Programático

### Integrar AURA en tu Aplicación

```python
from src.orchestrator import MultiAgentOrchestrator
from src.rag.vector_store import VectorStore
from src.config import config

# Inicializar
config.validate()
vector_store = VectorStore()
vector_store.load_vectorstore()

orchestrator = MultiAgentOrchestrator(vector_store)

# Iniciar sesión
greeting = orchestrator.start_session()
print(greeting)

# Simular respuestas del usuario
respuestas = [
    "1000 dólares",
    "Una laptop",
    "16GB RAM, procesador rápido",
    "Dell o Lenovo",
    "Programación"
]

# Procesar cada respuesta
for respuesta in respuestas:
    resultado = orchestrator.process_user_input(respuesta)
    print(resultado['message'])
    
    if resultado['status'] == 'completed':
        print("\n¡Recomendaciones generadas!")
        print(resultado['recommendations'])
        break

# Hacer pregunta de seguimiento
pregunta = "¿Cuál tiene mejor batería?"
resultado = orchestrator.process_user_input(pregunta)
print(resultado['message'])
```

## Tips para Mejores Resultados

### 1. Sé Específico en tus Respuestas
❌ "Una laptop"
✅ "Una laptop para edición de video 4K y gaming ocasional"

### 2. Menciona Prioridades
❌ "Buena batería y rendimiento"
✅ "Priorizo batería sobre rendimiento, viajo mucho"

### 3. Indica Limitaciones
❌ "No tengo preferencias"
✅ "No puedo cargar con más de 2kg, tengo problemas de espalda"

### 4. Contexto de Uso
❌ "Para trabajar"
✅ "Trabajo remoto 8 horas diarias, muchas videollamadas"

### 5. Experiencia Previa
❌ "Cualquier marca"
✅ "Tuve problemas con HP antes, prefiero evitarlos"
